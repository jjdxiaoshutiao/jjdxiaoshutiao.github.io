LocalPhysicsMode

enumeration

切换到手册

描述

提供 2D 和 3D 本地物理选项。

默认情况下，创建或加载 Scene 时，添加到场景中游戏对象的任何 2D 或 3D 物理组件都将添加到默认物理场景中。然而，每个 Scene 都能够创建和拥有自己的（本地）2D 和/或 3D 物理场景。创建或加载场景时，可以使用此选项指定是否应创建 2D 和/或 3D 物理场景。

当 Scene 创建自己的 2D 和/或 3D 物理场景时，物理场景的生命周期与 Scene 相同，即如果销毁/卸载 Scene ，那么也会销毁/卸载任何创建的物理场景。

另请参阅： CreateSceneParameters 、 LoadSceneParameters 、 SceneManager.CreateScene 、 SceneManager.LoadScene 和 SceneManager.LoadSceneAsync 。

变量

None 不会创建本地 2D 或 3D 物理场景。
Physics2D 本地 2D 物理场景将由 Scene 创建并拥有。
Physics3D 本地 3D 物理场景将由 Scene 创建并拥有。
