Assembly

class in UnityEditor.Compilation

切换到手册

描述

表示由 Unity 编译的程序集的类。

变量

allReferences 返回 Assembly.assemblyReferences 并合并 Assembly.compiledAssemblyReferences。构建该程序集时，这将返回传递给编译器的所有程序集。
assemblyReferences 用于构建此程序集的程序集引用。引用也是作为 Unity 项目的一部分而构建的程序集。另请参阅：Assembly.compiledAssemblyReferences 和 Assembly.allReferences。
compiledAssemblyReferences 用于构建此程序集的预编译程序集的程序集引用。另请参阅：Assembly.assemblyReferences 和 Assembly.allReferences。
compilerOptions 用于编译程序集的编译器选项。
defines 用于编译此程序集的定义。
flags 程序集的标志。另请参阅：AssemblyFlags。
name 程序集的名称。
outputPath 此程序集的完整输出文件路径。
sourceFiles 用于编译此程序集的所有源文件。
