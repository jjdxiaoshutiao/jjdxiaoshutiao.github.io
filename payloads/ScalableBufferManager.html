ScalableBufferManager

class in UnityEngine

切换到手册

描述

如果目标平台/图形 API 支持动态分辨率，缩放渲染纹理以支持动态分辨率。

调用 ResizeBuffers 时，ScalableBufferManager 处理标记为 DynamicallyScalable 的所有渲染纹理的缩放。标记为 DynamicallyScalable 的所有渲染纹理都按一个宽度和高度缩放因子缩放，通过缩放因子而不是使用特定的宽度和高度值控制缩放是因为不同的渲染纹理大小不同，但要按一个公共因子缩放。

静态变量

heightScaleFactor 用于控制动态分辨率的高度缩放因子。
widthScaleFactor 用于控制动态分辨率的宽度缩放因子。

静态函数

ResizeBuffers 用于调整标记为 DynamicallyScalable 的所有缓冲区大小的函数。
