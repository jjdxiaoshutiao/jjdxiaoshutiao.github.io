Application

class in UnityEngine.WSA

切换到手册

描述

提供与 Window Store 应用程序相关的基本方法。

静态变量

advertisingIdentifier 广告 ID。
arguments 传递给应用程序的参数。

静态函数

InvokeOnAppThread 在应用程序线程上执行回调项目。
InvokeOnUIThread 在 UI 线程上执行回调项目。
RunningOnAppThread 如果在应用程序线程上运行，返回 true。
RunningOnUIThread 如果在 UI 线程上运行，返回 true。

Events

windowActivated 激活应用程序窗口时触发。
windowSizeChanged 窗口大小改变时触发。
