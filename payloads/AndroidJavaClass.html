描述

AndroidJavaClass 是 java.lang.Class 的通用实例的 Unity 表示。

注意 ：此 API 可从自定义线程使用，但需要该线程首先附加到 JVM，请参阅 AndroidJNI.AttachCurrentThread 。

构造函数

AndroidJavaClass 基于类名称构造一个 AndroidJavaClass。

CallStatic 对类调用静态 Java 方法。
