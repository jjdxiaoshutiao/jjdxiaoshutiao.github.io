WSABuildAndRunDeployTarget

enumeration

切换到手册

描述

指定在编辑器中使用“Build and Run”时要在其上部署和启动此 UWP 应用的 Windows 设备。

执行常规构建生成 Visual Studio 解决方案时，将忽略此设置。

变量

LocalMachine 在本地 Windows PC 上运行此应用。
WindowsPhone 在通过 USB 连接到本地 PC 的 Windows 设备（例如 Windows Phone）上运行此应用。
DevicePortal 使用 Windows DevicePortal (WDP) 传输在远程 Windows 设备上运行此应用。
