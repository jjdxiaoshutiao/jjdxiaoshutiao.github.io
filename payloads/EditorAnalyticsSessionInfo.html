EditorAnalyticsSessionInfo

class in UnityEditor

切换到手册

描述

提供 Editor Analytics 会话信息的访问权限。

该类包含用于查找 Editor Analytics 系统收集的会话信息的静态方法。

静态变量

activeElapsedTime 自当前会话开始以来用户与 Editor 交互的总时间（以毫秒为单位）。
elapsedTime 当前会话的长度（以毫秒为单位）。
focusedElapsedTime Editor 在当前会话期间已获得焦点的总时间（以毫秒为单位）。
id 标识当前 Editor 会话的随机唯一 GUID。
playbackElapsedTime Editor 在当前会话期间已处于播放模式的总时间（以毫秒为单位）。
sessionCount 自安装 Unity Editor 的当前实例以来已产生的 Editor 会话数。
userId 唯一标识 Editor 安装的随机 GUID。
