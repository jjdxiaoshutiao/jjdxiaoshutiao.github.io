XRStats

class in UnityEngine.XR.Provider

切换到手册

描述

提供来自 XR 子系统的计时和其他统计信息。

XRStats 类允许 XR SDK 提供程序公开与其 XR 子系统相关的计时和其他统计信息。XR 应用程序开发者可以使用此类统计信息进行分析和动态性能调整。例如，应用程序可以在运行时动态调整 XRSettings.eyeTextureResolutionScale 或 XRSettings.renderViewportScale 等属性，以基于 XR 子系统提供的统计信息提高性能。

注意： XR SDK 提供程序可以使用此类提供其自己的、特定于设备的类来报告统计信息。XR 应用程序开发者应该不需要直接使用 XRStats 类。

静态函数

TryGetStat 获取 XR 子系统的统计信息。
