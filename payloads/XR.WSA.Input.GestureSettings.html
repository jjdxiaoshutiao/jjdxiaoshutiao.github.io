GestureSettings

enumeration

切换到手册

描述

此枚举表示 GestureRecognizer 可识别的一组手势。

GestureRecognizer 仅执行您请求的一组手势之间的最小消歧。例如，如果您仅请求点击，则用户可以在按住手指并持续任意时长，而且当用户释放手指时仍会发生点击。

变量

None 禁用对手势的支持。
Tap 启用对点击手势的支持。
DoubleTap 启用对双击手势的支持。
Hold 启用对长按手势的支持。
ManipulationTranslate 启用对跟踪手部位置变化的操作手势的支持。该手势相对于手势的开始位置，测量通过世界的绝对移动。
NavigationX 在水平轴上启用对导航手势的支持。
NavigationY 在垂直轴上启用对导航手势的支持。
NavigationZ 在深度轴上启用对导航手势的支持。
NavigationRailsX 使用 Rails（指南）在水平轴上启用对导航手势的支持。
NavigationRailsY 使用 Rails（指南）在垂直轴上启用对导航手势的支持。
NavigationRailsZ 使用 Rails（指南）在深度轴上启用对导航手势的支持。
