AssemblyReloadEvents

class in UnityEditor

切换到手册

描述

此类含有适用于程序集重载事件的事件分发程序。 

using UnityEditor;
using UnityEngine;

public class MyWindow : EditorWindow
{
    [MenuItem("Test/Show My Window")]
    static void Init()
    {
        GetWindow<MyWindow>();
    }

    void OnEnable()
    {
        AssemblyReloadEvents.beforeAssemblyReload += OnBeforeAssemblyReload;
        AssemblyReloadEvents.afterAssemblyReload += OnAfterAssemblyReload;
    }

    void OnDisable()
    {
        AssemblyReloadEvents.beforeAssemblyReload -= OnBeforeAssemblyReload;
        AssemblyReloadEvents.afterAssemblyReload -= OnAfterAssemblyReload;
    }

    public void OnBeforeAssemblyReload()
    {
        Debug.Log("Before Assembly Reload");
    }

    public void OnAfterAssemblyReload()
    {
        Debug.Log("After Assembly Reload");
    }
}

Events

afterAssemblyReload Unity 重新加载所有程序集后，会分发该事件。
beforeAssemblyReload Unity 重新加载所有程序集之前，会分发该事件。

委托

AssemblyReloadCallback 用于程序集重载事件的委托。
