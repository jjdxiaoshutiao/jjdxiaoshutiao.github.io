SubsystemManager

class in UnityEngine

切换到手册

描述

提供对子系统（通过插件提供额外功能）的访问权限。

提供查询 SubsystemDescriptor （枚举了功能）的功能。给定一个 SubsystemDescriptor ，您可以创建 Subsystem 来利用子系统。

静态函数

GetAllSubsystemDescriptors 不管特定的子系统类型为何，都将获取所有当前已知的子系统描述符。
GetInstances 返回特定实例类型的活动 Subsystem。
GetSubsystemDescriptors 返回 SubsystemDescriptor（描述了可启用的额外功能）的列表。

Events

reloadSubsytemsCompleted 完成重新加载所有 XR SDK Provider 打包的子系统后，从 SubsystemManager 调用。
reloadSubsytemsStarted 在重新加载所有 XR SDK Provider 打包的子系统前，从 SubsystemManager 调用。
