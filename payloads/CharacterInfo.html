CharacterInfo

struct in UnityEngine

切换到手册

描述

有关如何从字体纹理渲染字符的规范。请参阅 Font.characterInfo 。

另请参阅： Font.RequestCharactersInTexture 上的示例。

变量

advance 从该字符原点到下一个字符原点的水平距离，四舍五入到最接近的整数。
bearing 从该字形原点到该字形图像开头的水平距离。
glyphHeight 字形图像的高度。
glyphWidth 字形图像的宽度。
index 字符的 Unicode 值。
maxX 字形图像在 X 轴中的最大范围。
maxY 字形图像在 Y 轴中的最大范围。
minX 字形图像在 X 轴中的最小范围。
minY 字形图像在 Y 轴中的最小范围。
size 字符的大小，如果是默认字体大小，则为 0。
style 字符的样式。
uvBottomLeft 字体纹理中与字形图像左下部分匹配的 UV 坐标。
uvBottomRight 字体纹理中与字形图像右下部分匹配的 UV 坐标。
uvTopLeft 字体纹理中与字形图像左上部分匹配的 UV 坐标。
uvTopRight 字体纹理中与字形图像右上部分匹配的 UV 坐标。
