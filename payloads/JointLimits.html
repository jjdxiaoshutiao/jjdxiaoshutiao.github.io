JointLimits

struct in UnityEngine

切换到手册

描述

JointLimits 由 HingeJoint 用于限制关节角度。

另请参阅： HingeJoint 。 

using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        // Set the hinge limits for a door.
        HingeJoint hinge = GetComponent<HingeJoint>();

        JointLimits limits = hinge.limits;
        limits.min = 0;
        limits.max = 90;
        limits.bounciness = 0;
        limits.bounceMinVelocity = 0;
        hinge.limits = limits;
    }
}

变量

bounceMinVelocity 导致关节反弹的最小冲击速度。
bounciness 确定关节达到其限制时的反弹大小。也称为还原。
contactDistance 处于限制值内的距离，在该距离上解算器会将限制视为活动状态。
max 关节的角度上限（以度为单位）。
min 关节的角度下限（以度为单位）。
