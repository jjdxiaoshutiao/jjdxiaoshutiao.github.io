Column

class in UnityEditor.IMGUI.Controls

切换到手册

描述

列状态。

变量

allowToggleVisibility 用于允许/禁止从上下文菜单中隐藏列的选项。
autoResize 用于在调整整个 MultiColumnHeader 的大小时允许列自动调整大小的选项。
canSort 为此列启用了排序。如果设置为 false，则左键单击此列将不起作用。
contextMenuText 如果设置了此属性，则可以将其用于切换可见性的上下文菜单，如果未设置，则使用 ::headerContent。
headerContent 这是将在列标题中渲染的 GUIContent。
headerTextAlignment 对齐标题内容。
maxWidth 列的最大宽度。
minWidth 列的最小宽度。
sortedAscending 控制此列是按升序还是按降序排序的值。
sortingArrowAlignment 对齐排序箭头。
width 列的宽度。
