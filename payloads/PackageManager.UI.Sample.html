描述

包样本的结构。

变量

description 包样本的描述。
displayName 包样本的显示名称。
importPath 在项目资源文件夹下，将样本导入到的完整路径。
interactiveImport 指示导入资源包（.unitypackage 文件）样本时是否显示导入窗口。
isImported 指示是否已导入了该样本。
resolvedPath 样本在磁盘上的完整路径，在包含样本的包内。

公共函数

Import 将包样本导入 "Assets" 文件夹。

静态函数

FindByPackage 查找基于特定版本的包中的示例列表。
