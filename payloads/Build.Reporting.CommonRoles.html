CommonRoles

class in UnityEditor.Build.Reporting

切换到手册

描述

该类为构建中的文件使用的一些常见角色提供常量值。构建中的每个文件的角色可在 BuildFile.role 中找到。

静态变量

appInfo 提供在 Windows 低完整性模式下使用的配置信息的文件的 BuildFile.role 值。
assetBundle 构建的 AssetBundle 文件的 BuildFile.role 值。
assetBundleTextManifest AssetBundle 清单文件的 BuildFile.role 值，该文件在构建过程中生成且包含有关捆绑包及其依赖关系的信息。
bootConfig 包含引擎启动早期阶段的配置信息的文件的 BuildFile.role 值。
builtInResources 包含引擎内置资源的文件的 BuildFile.role 值。
builtInShaders 包含标准着色器等 Unity 内置着色器的文件的 BuildFile.role 值。
crashHandler 用于捕获播放器崩溃情况的可执行文件的 BuildFile.role 值。
debugInfo 包含调试器信息的文件的 BuildFile.role 值。
dependentManagedLibrary 托管库的 BuildFile.role 值，该库由于是 CommonRoles.managedLibrary 的依赖项而存在于构建中。
engineLibrary 当主 Unity 运行时作为单独的库构建时的 BuildFile.role 值。
executable 可执行文件的 BuildFile.role 值 - 将在目标设备上实际启动的文件。
globalGameManagers 包含播放器的全局项目设置数据的文件的 BuildFile.role 值。
managedEngineApi 为 Unity 提供托管 API 的文件的 BuildFile.role 值。
managedLibrary 包含编译脚本代码的托管程序集的 BuildFile.role 值。
manifestAssetBundle 清单 AssetBundle 的 BuildFile.role 值，该 AssetBundle 包含有关其他 AssetBundle 及其依赖关系的信息。
monoConfig 由 Mono 运行时用作配置数据的文件的 BuildFile.role 值。
monoRuntime 构成 Mono 运行时的文件的 BuildFile.role 值。
resourcesFile 包含项目“资源”文件夹中内容（打包到单个文件中）的文件的 BuildFile.role 值。
scene 包含场景打包内容的文件的 BuildFile.role 值。
sharedAssets 包含场景之间共享的资源对象的文件的 BuildFile.role 值。资源对象示例包括纹理、模型和音频。
streamingAsset 在未修改项目中 StreamingAssets 文件夹的情况下复制到构建中的文件的 BuildFile.role 值。
streamingResourceFile 包含流媒体资源数据的文件的 BuildFile.role 值。
