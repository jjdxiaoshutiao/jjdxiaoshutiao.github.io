描述

此类包含绘制与播放器连接相关的 IMGUI 编辑器 UI 的方法。

使用此类中的方法手动定位 UI 元素。要自动进行 UI 布局，请使用 EditorGUILayout 中的相应方法。

静态函数

AttachToPlayerDropdown 显示下拉选单按钮和菜单，供用户选择并建立与播放器的连接。
