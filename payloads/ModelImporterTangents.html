ModelImporterTangents

enumeration

切换到手册

描述

ModelImporter 的顶点切线生成选项。

切线可以从模型文件导入，可以由 Unity 使用不同方法计算（默认方法为 MikkTSpace），也可以不包含在导入网格中。顶点切线最常用于法线贴图/凹凸贴图。

另请参阅： ModelImporter.importTangents 、 ModelImporterNormals 、 Mesh.tangents 。

变量

Import 从模型文件中导入顶点切线。
CalculateLegacy 使用旧版算法计算切线。
CalculateLegacyWithSplitTangents 使用旧版算法计算切线，并在 UV 图表上进行拆分。
CalculateMikk 使用 MikkTSpace 计算切线（默认设置）。
None 不导入顶点切线。
