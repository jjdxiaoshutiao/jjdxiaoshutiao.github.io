IShouldIncludeInBuildCallback

interface in UnityEditor.PackageManager

切换到手册

描述

包管理器用来请求有关包的构建信息的接口。

在构建操作中，对包中名为 PackageName 的每个托管插件 (DLL) 调用 ShouldIncludeInBuild 方法。

使用 BuildUtilities.RegisterShouldIncludeInBuildCallback 在 Package Manager 中注册你的实现。只能为每个包注册此接口的一个实现。

另请参阅： BuildUtilities 。

变量

PackageName 包的名称。

公共函数

ShouldIncludeInBuild 确定是否在构建中包含托管插件。
