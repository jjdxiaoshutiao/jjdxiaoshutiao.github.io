LookAtConstraint

class in UnityEngine.Animations

/

继承自： Behaviour


实现接口： IConstraint

切换到手册

描述

约束一个对象相对于一个或多个源对象所在位置的方向，例如对象面向源对象的平均位置。 LookAtConstraint 是一个简化的 AimConstraint ，通常与 Camera 一起使用。

变量

constraintActive 激活或停用约束。
locked 锁定静止的偏移和旋转。
roll 沿对象 z 轴的旋转角度。当 Animations.LookAtConstraint.UseUpObject 为 false 时，约束使用此属性计算世界向上矢量。
rotationAtRest 当源的总权重为 0 时使用的旋转。
rotationOffset 表示相对于受约束方向的偏移。
sourceCount 组件中设置的源的数量（只读）。
useUpObject 确定如何计算向上矢量。
weight 约束组件的权重。
worldUpObject 世界向上对象，用于在 Animations.LookAtConstraint.UseUpObject 为 true 时计算世界向上矢量。
