ProfilerArea

enumeration

切换到手册

描述

分析的不同区域，对应于 ProfilerWindow 中的图表。

每个区域对应于 ProfilerWindow 中的图表以及其下半部分中随附的详细信息面板。这些是对统计数据进行分组并用于深入了解这些特定区域的类别。

变量

CPU CPU 统计信息。
GPU GPU 统计信息。
Rendering 渲染统计信息。
Memory 内存统计信息。
Audio 音频统计信息。
Video 视频回放统计信息。
Physics 3D 物理统计信息。
Physics2D 2D 物理统计信息。
NetworkMessages 网络消息统计信息。
NetworkOperations 网络操作统计信息。
UI UI 统计信息。
UIDetails 详细 UI 统计信息。
GlobalIllumination 全局光照统计信息。
