Il2CppCompilerConfiguration

enumeration

切换到手册

描述

编译 IL2CPP 生成的代码时使用的 C++ 编译器配置。

变量

Debug Debug 配置停用了所有优化，这使得代码构建速度更快，但运行速度变慢。
Release Release 配置启用了优化，因此编译后的代码运行速度更快，二进制文件大小变小，但编译时间变长。
Master Master 配置支持所有可能的优化，淋漓尽致发挥出全部的性能。例如，在使用 MSVC++ 编译器的平台上，此选项允许在链接时生成代码。使用此配置编译代码所需的时间比使用 Release 配置要长得多。如果构建时间的增加量在可接受范围，Unity 建议使用 Master 配置构建游戏的发布版本。
