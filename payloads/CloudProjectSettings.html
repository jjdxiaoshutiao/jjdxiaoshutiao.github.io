CloudProjectSettings

class in UnityEditor

切换到手册

描述

使用此类检索关于当前选定的项目和当前已登录的 Unity ID 的信息。

静态变量

organizationId 此项目所属的组织的 ID。（只读）
organizationName 此项目所属的组织的名称。（只读）
projectId 唯一的云端项目标识符。它对于每个项目而言都是唯一的（只读）。
projectName 控制面板中与此项目相关联的项目条目名称（只读）。
userId 当前已登录的 Unity ID 帐户的用户 ID（只读）。
userName 当前已登录的 Unity ID 帐户的用户名（只读）。
