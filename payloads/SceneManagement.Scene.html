Scene

struct in UnityEngine.SceneManagement

切换到手册

描述

*.unity 文件的运行时数据结构。

变量

buildIndex 返回 Build Settings 中场景的索引。
isDirty 如果场景已修改，则返回 true。
isLoaded 如果场景已加载，则返回 true。
name 返回游戏或应用程序中当前处于活动状态的场景的名称。
path 返回场景的相对路径。如：“Assets/MyScenes/MyScene.unity”。
rootCount 此场景的根变换组件的数量。

公共函数

GetRootGameObjects 返回场景中的所有根游戏对象。
IsValid 这是否为有效场景。 例如，如果您尝试打开某个不存在的场景，则该场景为无效场景。在这种情况下，从 EditorSceneManager.OpenScene 返回的场景会因 IsValid 返回 false。

运算符

operator != 如果场景不同，则返回 true。
operator == 如果场景相同，则返回 true。
