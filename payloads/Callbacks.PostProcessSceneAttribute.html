PostProcessSceneAttribute

class in UnityEditor.Callbacks

/

继承自： CallbackOrderAttribute

切换到手册

描述

构建场景后，将此属性添加至方法以接收通知。

调用 SceneManager.LoadScene 时，系统会在进入播放模式时调用带有此属性的方法。

PostProcessSceneAttribute 具有提供回调中的顺序索引的选项，从 0 开始。如果您有多个 OnPostprocessScene 回调并想以特定顺序调用它们，此选项将非常有用。系统将从零开始按顺序调用回调。 

// C# example:
// Automatically creates a game object with a primitive mesh renderer and appropriate collider.
using UnityEngine;
using UnityEditor;
using UnityEditor.Callbacks;

public class MyScenePostprocessor {
    [PostProcessSceneAttribute (2)]
    public static void OnPostprocessScene() {
        GameObject cube = GameObject.CreatePrimitive(PrimitiveType.Cube);
        cube.transform.position = new Vector3(0.0f, 0.5f, 0.0f);
    }
}
