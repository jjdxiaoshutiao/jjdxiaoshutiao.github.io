XRInputSubsystem

class in UnityEngine.XR

/

继承自： IntegratedSubsystem

切换到手册

描述

XRInputSubsystem 实例用于启用和禁用来自特定插件的输入。

启动一个 XRInputSubsystem 实例将会导致该插件开始将输入设备数据馈送至以下 InputTracking 系统：XR.InputTracking.GetLocalPosition 和 XR.InputTracking.GetLocalRotation。调用 Stop 或 Shutdown 函数将会禁用对该插件的任何输入设备数据的轮询。

公共函数

GetSupportedTrackingOriginModes 获取此子系统支持的所有 TrackingOriginModeFlags。
GetTrackingOriginMode 获取跟踪原点模式。
TryGetBoundaryPoints 获取表示 SDK 设置边界的 3D 位置值的列表。
TryGetInputDevices 获取此 XRInputSubsystem 报告的所有连接的 InputDevice 的列表。
TryRecenter 将所有 InputDevice 上的跟踪功能居中到头戴式设备的当前位置和方向。
TrySetTrackingOriginMode 尝试设置子系统的 TrackingOriginModeFlags。

Events

boundaryChanged 一个事件，接收 XRInputSubsystem 在更改其跟踪边界时调用的委托实例。
trackingOriginUpdated 一个事件，接收 XRInputSubsystem 在更改其报告设备所在的原点位置时调用的委托实例。

继承的成员

变量

公共函数

Destroy 销毁子系统的此实例。
Start 启动子系统的一个实例。
Stop 停止子系统的一个实例。
