GameViewRenderMode

enumeration

切换到手册

描述

在 Game 视图中或主机 PC 上的主窗口中枚举可用的 XR 渲染模式。仅当 Unity Editor 处于播放模式时才会进行 XR 渲染。

变量

None 在 Game 视图中或主机 PC 上的主窗口中禁用任何眼睛的渲染。
LeftEye 在 Game 视图窗口中或主机 PC 上的主窗口中渲染 XR 设备的左眼。
RightEye 在 Game 视图窗口中或主机 PC 上的主窗口中渲染 XR 设备的右眼。
BothEyes 在 Game 视图中或主机 PC 上的主窗口中并排渲染 XR 设备的双眼。
OcclusionMesh 在 Game 视图中或主机 PC 上的主窗口中并排渲染 XR 设备的双眼以及遮挡网格。
