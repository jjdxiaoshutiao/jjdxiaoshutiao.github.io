AnalyticsSettings

class in UnityEditor.Analytics

切换到手册

描述

适用于 Unity 服务编辑器功能的 Editor API。Analytics 通常从 Services 窗口启用，但如果您要编写自己的编辑器扩展，则可以使用该 API。

静态变量

configUrl 设置 Analytics 配置终端。
deviceStatsEnabledInBuild 报告是否在运行时启用设备统计信息。
enabled 如果设置为 true，该布尔值字段启用 Unity 中的 Analytics 功能。如果设置为 false，则禁用该功能。
eventUrl 设置 Analytics 事件终端。
initializeOnStartup 控制 Unity 是否在启动时立即初始化 Analytics。
testMode 设置为 true 以仅在编辑器中测试 Analytics 集成。

Events

OnRequireInBuildHandler 只要平台版本启动都进行分发。使用此事件可在平台版本中启用 Analytics。

委托

RequireInBuildDelegate 定义用于处理 AnalyticsSettings.RequireInBuildDelegate 事件的委托签名。
