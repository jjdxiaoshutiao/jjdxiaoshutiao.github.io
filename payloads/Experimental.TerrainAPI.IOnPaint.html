IOnPaint

interface in UnityEditor.Experimental.TerrainAPI

切换到手册

描述

此接口为地形绘制工具的 OnPaint 事件提供参数和实用函数。

另请参阅： TerrainPaintTool<T0>.OnPaint 。

变量

brushSize 只读。采用地形单位的当前笔刷大小（相当于世界单位的大小）。
brushStrength 只读。当前画笔强度。
brushTexture 只读。当前所选的画笔纹理。
hitValidTerrain 只读。如果鼠标位于有效的地形对象上方，则为 true；否则为 false。
raycastHit 只读。当前鼠标位置的射线投射结果。这在 hitValidTerrain 为 true 时有效。
uv 只读。激活的地形上的标准化位置（介于 0 和 1 之间）。

公共函数

Repaint 指示编辑器重绘工具 UI 和/或 Scene 视图。
