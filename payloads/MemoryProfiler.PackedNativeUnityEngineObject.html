PackedNativeUnityEngineObject

struct in UnityEditor.MemoryProfiler

切换到手册

描述

内存中 C++ unity 对象的描述。

变量

hideFlags 此本机对象拥有的 hideFlags。
instanceId 此对象的 InstanceId。
isDontDestroyOnLoad 是否将此对象标记为 DontDestroyOnLoad？
isManager 此本机对象是否为内部 Unity 管理器对象？
isPersistent 此对象是否持久存在？（资源以及场景中存储的对象具有持久性，动态创建的对象则不具有持久性）
name 此对象的名称。
nativeObjectAddress 本机 C++ 对象的内存地址。这与 UnityEngine.Object 的“m_CachedPtr”字段相匹配。
nativeTypeArrayIndex 用于从 PackedMemorySnapshot.nativeTypes 数组获取本机 C++ 类型描述的索引。
size 此对象的大小（以字节为单位）。
