XRStats

class in UnityEngine.XR

切换到手册

描述

XR 子系统的用时和其他统计信息。

一些 XR SDK 提供对额外用时和其他统计信息的访问。游戏和应用程序可使用这些信息进行分析和动态性能调整。例如，在运行时修改 XRSettings.eyeTextureResolutionScale 或 XRSettings.renderViewportScale 可以提高性能。此类公开了一组可由 SDK 选择性报告的信息。请务必使用任何方法的返回值来了解 SDK 是否正在报告数据。

静态函数

TryGetDroppedFrameCount 检索 XR SDK 报告的丢弃的帧数。
TryGetFramePresentCount 检索 XR SDK 报告的当前帧绘制到设备中的次数。
TryGetGPUTimeLastFrame 检索 XR SDK 报告的 GPU 最后一帧花费的时间（以秒为单位）。
