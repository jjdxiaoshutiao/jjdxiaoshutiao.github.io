SettingsScope

enumeration

切换到手册

描述

设置 SettingsProvider 的作用域。“作用域”确定了其在 UI 中的显示位置。例如，是与 Settings 窗口还是 Preferences 窗口中的项目设置 (Project settings) 一起显示，或在两个窗口中都显示。

另请参阅： SettingsProvider.scope 、 SettingsProvider.SettingsProvider 。

变量

User SettingsProvider 仅显示在 Preferences 窗口中。
Project SettingsProvider 仅显示在 Project Settings 窗口中。
