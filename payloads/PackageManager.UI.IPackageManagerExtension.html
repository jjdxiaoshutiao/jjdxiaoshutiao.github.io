IPackageManagerExtension

interface in UnityEditor.PackageManager.UI

切换到手册

描述

Package Manager UI 扩展的接口。

公共函数

CreateExtensionUI 创建扩展 UI 可视元素。
OnPackageAddedOrUpdated 添加或更新包时由 Package Manager UI 调用。
OnPackageRemoved 移除包时由 Package Manager UI 调用。
OnPackageSelectionChange 更改包的选择内容时由 Package Manager UI 调用。
