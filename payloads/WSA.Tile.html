Tile

class in UnityEngine.WSA

切换到手册

描述

表示 Windows 开始屏幕上的瓦片

此类可以用于创建或更新开始屏幕上的辅助瓦片，以及获取瓦片实例以向它们发送通知。

静态变量

main 返回应用程序主瓦片

变量

exists 辅助瓦片是否固定到开始屏幕。
hasUserConsent 用户是批准（固定到开始屏幕）还是拒绝了辅助瓦片。
id 唯一字符串，标识辅助瓦片

公共函数

Delete 显示从开始屏幕取消固定辅助瓦片的请求。
PeriodicBadgeUpdate 启动瓦片上的标示的定期更新。
PeriodicUpdate 启动瓦片的定期更新。
RemoveBadge 从瓦片中删除标示。
StopPeriodicBadgeUpdate 停止之前启动的瓦片定期更新。
StopPeriodicUpdate 停止之前启动的瓦片定期更新。
Update 为瓦片发送通知（更新瓦片外观）。
UpdateBadgeImage 将瓦片上的标示设置或更新为图像。
UpdateBadgeNumber 将瓦片上的标示设置或更新为数字。

静态函数

CreateOrUpdateSecondary 创建新的或更新现有的辅助瓦片。
DeleteSecondary 显示从开始屏幕取消固定辅助瓦片的请求。
Exists 辅助瓦片是否固定到开始屏幕。
GetSecondaries 获取所有辅助瓦片。
GetSecondary 返回辅助瓦片（由瓦片 ID 标识）。
GetTemplate 获取瓦片通知的模板 XML。
