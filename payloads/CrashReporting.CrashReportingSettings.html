CrashReportingSettings

class in UnityEditor.CrashReporting

切换到手册

描述

适用于 Unity 服务编辑器功能的 Editor API。通常使用 Services 窗口启用 CrashReporting，但如果您要编写自己的编辑器扩展，可以使用此 API。

静态变量

captureEditorExceptions 此布尔值字段会影响 Unity 中的 CrashReporting 功能：如果为 true，则此功能会在运行播放模式时捕获编辑器中出现的异常；如果为 false，则忽略所有这些错误。
enabled 此布尔值字段决定是否启用 Unity 中的 CrashReporting 功能：如果为 true，则启用；如果为 false，则禁用。
logBufferSize Performance Reporting 服务将保留近期最多 X 条日志消息（Debug.Log 等）的缓冲内容与崩溃报告一起发送。默认为 10 条日志消息，最大值为 50。将此值设置为 0 可禁用崩溃报告日志捕获。
