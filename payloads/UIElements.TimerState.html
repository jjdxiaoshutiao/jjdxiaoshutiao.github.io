TimerState

struct in UnityEngine.UIElements

切换到手册

描述

包含计划程序事件的计时信息。

变量

deltaTime now 与上一个回调之间的时间差（以毫秒为单位）。
now 当前时间（以毫秒为单位）。
start 开始时间（以毫秒为单位），或可重复 IScheduledItem 的上次回调时间。

公共函数

Equals 将此对象与另一个对象进行比较，如果相等，则返回 true。

运算符

operator != 比较两个 TimerState 对象，如果不相等，则返回 true。
operator == 比较两个 TimerState 对象，如果相等，则返回 true。
