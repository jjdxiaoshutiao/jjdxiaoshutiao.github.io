MediaRational

struct in UnityEditor.Media

切换到手册

描述

有理数对于精确表示分数很有用。

静态变量

Invalid 无效有理值。

变量

denominator 分数分母。
inverse 有理数的倒数。
isNegative 有理数是否为负值。
isValid 有理数是否有效。
isZero 有理数是否为零。
numerator 分数分子。

构造函数

MediaRational 构造一个有理数。忽略分母的版本将其设置为 1。

公共函数

Set 设置分子和分母，从而执行归一化。

运算符

Unknown operator 返回表示为浮点值的有理值。
