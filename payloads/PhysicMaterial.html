PhysicMaterial

class in UnityEngine

/

继承自： Object

切换到手册

描述

物理材质描述如何处理碰撞对象（摩擦力、弹性）。

另请参阅： Collider 。

变量

bounceCombine 确定如何组合弹性。
bounciness 表面的弹性如何？值为 0 将不会反弹。值为 1 将在反弹时无任何能量损失。
dynamicFriction 已在移动时使用的摩擦力。此值通常介于 0 到 1 之间。
frictionCombine 确定如何组合摩擦力。
staticFriction 当对象处于表面上时使用的摩擦系数。

PhysicMaterial 创建新材质。
