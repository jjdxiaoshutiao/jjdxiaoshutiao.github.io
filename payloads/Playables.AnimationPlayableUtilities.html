AnimationPlayableUtilities

class in UnityEngine.Playables

切换到手册

描述

实现高级 utility 方法以简化带有动画的 Playable API 的使用。

静态函数

Play 在给定的动画器上播放 Playable。
PlayAnimatorController 创建要在给定动画器上播放的 PlayableGraph。此外，还为给定 RuntimeAnimatorController 创建一个 AnimatorControllerPlayable。
PlayClip 创建要在给定动画器上播放的 PlayableGraph。此外，还为给定 AnimationClip 创建一个 AnimationClipPlayable。
PlayLayerMixer 创建要在给定动画器上播放的 PlayableGraph。此外，还创建一个 AnimationLayerMixerPlayable。
PlayMixer 创建要在给定动画器上播放的 PlayableGraph。此外，还创建一个 AnimationMixerPlayable。
