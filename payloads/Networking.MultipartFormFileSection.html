MultipartFormFileSection

class in UnityEngine.Networking


实现接口： IMultipartFormSection

切换到手册

描述

用于通过 [IMultipartFormSection] API 将文件上载添加到多部分表单的 helper 对象。

此对象与 [MultipartFormDataSection] 对象非常相似，但所有构造函数另外还接受（且需要）一个 fileName 。如果您忽略 fileName 参数，此对象将提供默认文件名。

变量

contentType 返回该分区的 Content-Type 标头的值。
fileName 返回表示目标服务器上此节所需文件名的字符串。
sectionData 返回此节中包含的原始二进制数据。不得返回 null 或零长度数组。
sectionName 返回此节的名称（如果有）。

构造函数

MultipartFormFileSection 包含一个已命名文件分区，该分区基于 data 中的原始字节且具有自定义 Content-Type 和文件名。
