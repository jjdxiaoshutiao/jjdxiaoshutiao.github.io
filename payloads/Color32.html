Color32

struct in UnityEngine

切换到手册

描述

32 位格式的 RGBA 颜色表示形式。

每个颜色分量都是范围是 0 至 255 的字节值。

分量 ( r , g , b ) 定义 RGB 颜色空间中的颜色。Alpha 分量 ( a ) 定义透明度 - alpha 为 255 表示完全不透明，alpha 为零表示完全 透明。

变量

a 颜色的 Alpha 分量。
b 颜色的蓝色分量。
g 颜色的绿色分量。
r 颜色的红色分量。
this[int] 分别使用 [0]、[1]、[2]、[3] 访问红色 (r)、绿色 (g)、蓝色 (b) 和、Alpha (a) 颜色分量。

构造函数

Color32 使用给定的 r、g、b、a 分量构造新的 Color32。

公共函数

ToString 返回该颜色的整齐格式化的字符串。

静态函数

Lerp 在颜色 a 与 b 之间按 t 进行线性插值。
LerpUnclamped 在颜色 a 与 b 之间按 t 进行线性插值。

运算符

Color Color32 可与 Color 隐式相互转换。
Color32 Color32 可与 Color 隐式相互转换。
