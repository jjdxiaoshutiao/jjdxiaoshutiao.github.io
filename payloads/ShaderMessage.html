ShaderMessage

struct in UnityEditor

切换到手册

描述

包含 Unity 的着色器编译器所生成消息的相关信息。

变量

file 发现着色器消息所在的源文件。
line 发现着色器消息所在的代码行。
message Unity 着色器编译器返回的消息。
messageDetails Unity 着色器编译器返回的（可选）详细消息。
platform Unity 着色器编译器发现消息所在的特定平台。
severity 指示 Unity 着色器编译器所返回消息的严重性。

构造函数

ShaderMessage 使用给定消息和严重性创建新 ShaderMessage。

公共函数

Equals 逐个元素对两个 ShaderMessage 进行比较。
GetHashCode 根据消息的内容生成哈希值。

运算符

operator != 如果两个参数的任何元素不相等，则返回 true。
operator == 如果两个参数的所有成员都相等，则返回 true。
