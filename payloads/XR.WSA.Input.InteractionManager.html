InteractionManager

class in UnityEngine.XR.WSA.Input

切换到手册

描述

提供对用户通过手、控制器和系统语音命令执行的输入的访问权限。

Events

InteractionSourceDetected 检测到新的手、控制器或语音命令源时发生。
InteractionSourceDetectedLegacy 检测到新的手、控制器或语音命令源时发生。
InteractionSourceLost 当手、控制器或语音命令源不再可用时发生。
InteractionSourceLostLegacy 当手、控制器或语音命令源不再可用时发生。
InteractionSourcePressed 当手或控制器进入按下状态时发生。
InteractionSourcePressedLegacy 当手或控制器进入按下状态时发生。
InteractionSourceReleased 当手或控制器退出按下状态时发生。
InteractionSourceReleasedLegacy 当手或控制器退出按下状态时发生。
InteractionSourceUpdated 当手或控制器遇到 SourceState 更改时发生。
InteractionSourceUpdatedLegacy 当手或控制器遇到 SourceState 更改时发生。

委托

SourceEventHandler 回调以处理 InteractionManager 事件。
