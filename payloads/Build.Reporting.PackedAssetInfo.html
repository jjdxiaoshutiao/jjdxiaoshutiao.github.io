PackedAssetInfo

struct in UnityEditor.Build.Reporting

切换到手册

描述

包含有关单个打包资源的信息。

变量

id 打包资源的唯一标识符。
offset 指示打包资源开始的 PackedAssets 文件中的偏移。
packedSize 打包资源的大小。
sourceAssetGUID 构建过程用于生成打包资源的源资源的全局唯一标识符 (GUID)。
sourceAssetPath 构建过程用于生成打包资源的源资源相对于 Project 目录的文件路径。
type 构建过程用于生成打包资源的源资源的类型，例如图像、网格或音频类型。
