AreaScope

class in UnityEngine

切换到手册

描述

用于管理 BeginArea / EndArea 的可处置的 Helper 类。

BeginArea 在构造时调用， EndArea 在处置实例时调用。 默认情况下，使用 GUILayout 创建的任何 GUI 控件都放置在屏幕的左上角。 若要在任意区域放置一系列自动布局的控件，请使用 GUILayout.BeginArea 为自动布局系统定义要使用的新区域。

另请参阅： BeginArea 、 EndArea

\ 说明示例区域。 

using UnityEngine;

public class ExampleClass : MonoBehaviour
{
    void OnGUI()
    {
        using (var areaScope = new GUILayout.AreaScope(new Rect(10, 10, 100, 100)))
        {
            GUILayout.Button("Click me");
            GUILayout.Button("Or me");
        }
    }
}

构造函数

GUILayout.AreaScope 创建一个新的 AreaScope 并开始相应的区域。
