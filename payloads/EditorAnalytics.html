EditorAnalytics

class in UnityEditor

切换到手册

描述

适用于 EditorAnalytics 功能的 Editor API。

静态变量

enabled 如果已启用 EditorAnalytics，则返回 true。

静态函数

RegisterEventWithLimit 此 API 用于注册 Editor Analytics 事件。注意：此 API 仅供内部使用，未来可能会发生更改。不要在用户代码中使用。
SendEventWithLimit 此 API 用于发送 Editor Analytics 事件。注意：此 API 仅供内部使用，未来可能会发生更改。不要在用户代码中使用。
SetEventWithLimitEndPoint 使用此 API 可设置事件终端 URL。注意：此 API 仅供内部使用，未来可能会发生更改。不要在用户代码中使用。
SetEventWithLimitPriority 使用此 API 可设置事件优先级。注意：此 API 仅供内部使用，未来可能会发生更改。不要在用户代码中使用。
