IMultipartFormSection

interface in UnityEngine.Networking

切换到手册

描述

用于将数据组合到多部分表单中的接口。

为了让需要生成多部分表单数据的用户能够实现更精确的控制，同时使大多数用户无需参照 RFC 2388 ，Unity 提供了这个简单的接口，UnityWebRequest API 可使用该接口将复杂的数据序列化为正确格式的字节。

为方便起见，我们已将这两种常规类型的表单节封装到 IMultipartFormSection 的两个 stock 实现中。只需使用构造函数即可控制这两个 stock 实现。

通过 [UnityWebRequest.SerializeFormSections] 将 IMultipartFormSection 实现器转换为字节。

另请参阅：[MultipartFormDataSection]、[MultipartFormFileSection]。

变量

contentType 返回要在此表单节的 Content-Type 标头中使用的值。
fileName 返回表示目标服务器上此节所需文件名的字符串。
sectionData 返回此节中包含的原始二进制数据。不得返回 null 或零长度数组。
sectionName 返回此节的名称（如果有）。
