描述

截屏的原始数据表示形式。

表示形式由性能分析 API 内部数据和相关截屏的数据填充。可以使用此类型提取截屏数据。

变量

height 图像的高度。
imageFormat 图像捕获格式。
rawImageDataReference 对图像数据的非拥有引用。
width 图像的宽度。
