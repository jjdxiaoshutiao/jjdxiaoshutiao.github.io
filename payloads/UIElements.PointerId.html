PointerId

class in UnityEngine.UIElements

切换到手册

描述

保存指针 ID 值的静态类。

这些值用作 IPointerEvent.pointerId 的值。

静态变量

invalidPointerId 表示无效的指针 ID 值。
maxPointers 实现支持的最大指针数。
mousePointerId 鼠标指针 ID。
penPointerCount 实现支持的笔指针的数量。
penPointerIdBase 笔指针的基本 ID。
touchPointerCount 实现支持的触摸指针的数量。
touchPointerIdBase 触摸指针的基本 ID。
