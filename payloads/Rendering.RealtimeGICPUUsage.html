RealtimeGICPUUsage

enumeration

切换到手册

描述

在运行时为最终光照计算分配多少 CPU 使用率。

在 Player 中为 Realtime Global Illumination 光照计算创建的 CPU 工作线程数。增加此值会加快系统对光照更改的反应速度，成本是使用更多 CPU 时间。CPU Usage 值越高，为解决 Realtime GI 创建的工作线程就越多。\ \ 请注意 有些平台允许工作线程占用所有 CPU，而有些平台有最大限制：\ Xbox One： 4 个 CPU 内核。\ PS4： 4 个 CPU 内核。\ Android： 如果设备为 bigLittle 架构，将只使用小 CPU，否则为 CPU 数 - 1。\

变量

Low 将可用 CPU 线程的 25% 用作工作线程
Medium 将可用 CPU 线程的 50% 用作工作线程
High 将可用 CPU 线程的 75% 用作工作线程。
Unlimited 将可用 CPU 线程的 100% 用作工作线程
