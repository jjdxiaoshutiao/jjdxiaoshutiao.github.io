Asset

class in UnityEditor.VersionControl

切换到手册

描述

此类包含有关资源的版本控制状态的信息。

变量

assetPath 获取相对于项目根的资源路径。如果资源是元文件，则返回该元文件的路径。
fullName 获取资源的全名，包括扩展名。
isFolder 如果资源是文件夹，则返回 true。
isInCurrentProject 如果资源文件存在并位于当前项目中，则返回 true。
isMeta 如果 Asset 类的实例实际上会引用 .meta 文件，则返回 true。
locked 如果资源被版本控制系统锁定，则返回 true。
metaPath 获取此资源相对于项目根的元文件路径。如果资源是元文件，则返回该元文件的路径。
name 获取资源的名称。
path 获取资源的路径。
readOnly 如果资源是只读的，则返回 true。
state 获取资源的版本控制状态。

公共函数

Edit 在关联的编辑器中打开资源。
IsOneOfStates 如果资源的版本控制状态是输入状态之一，则返回 true。
IsState 如果资源的版本控制状态与输入状态完全一致，则返回 true。
Load 将资源加载到内存中。
