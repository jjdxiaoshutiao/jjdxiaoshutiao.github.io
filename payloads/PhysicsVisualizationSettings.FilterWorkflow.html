FilterWorkflow

enumeration

切换到手册

描述

决定 Physics Debug 窗口中的 Workflow 应为 inclusive/ ( ShowSelectedItems ) 还是 exclusive ( HideSelectedItems )。

默认选择 HideSelectedItems ，且不选择任何过滤项。这意味着未隐藏任何内容，所有内容均可见。

另请参阅：PhysicsVisualizationSettings.filterWorkflow。

变量

HideSelectedItems 使用 HideSelectedItems，您可以隐藏选定的过滤项，以便轻松丢弃不感兴趣的碰撞体属性，从而更轻松地进行概述和导航。
ShowSelectedItems 使用 ShowSelectedItems 和 Select Nonex 按钮，您可以轻松监控非常具体的碰撞体。
