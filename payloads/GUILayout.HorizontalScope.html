HorizontalScope

class in UnityEngine

切换到手册

描述

用于管理 BeginHorizontal / EndHorizontal 的可处置的 Helper 类。

在该元素内呈现的所有控件都将依次水平放置。 using 语句意味着不需要 BeginHorizontal 和 EndHorizontal 。

\ 水平布局。 

using UnityEngine;

public class ExampleClass : MonoBehaviour
{
    void OnGUI()
    {
        // Starts a horizontal group
        using (var horizontalScope = new GUILayout.HorizontalScope("box"))
        {
            GUILayout.Button("I'm the first button");
            GUILayout.Button("I'm to the right");
        }
        // Now the group is ended.
    }
}

构造函数

GUILayout.HorizontalScope 创建一个新的 HorizontalScope 并开始相应的水平组。
