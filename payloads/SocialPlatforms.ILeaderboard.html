ILeaderboard

interface in UnityEngine.SocialPlatforms

切换到手册

描述

排行榜包含特定游戏中所有玩家的得分。

每个游戏可有多个得分不同的排行榜。可自定义排行榜对象以执行特定查询。排行榜 ID 定义了正在查询哪个排行榜，并且可使用过滤器缩小结果范围： UserScope 、 TimeScope 、 Range 以及 SetUserFilter 。 使用 Social.CreateLeaderboard 创建此对象的实例。

变量

id 此排行榜的唯一标识符。
loading 此排行榜正在加载得分。
localUserScore 已登录用户的排行榜得分。
maxRange 排行榜包含的总得分。
range 此排行榜返回的等级范围。
scores 查询返回的排行榜得分。
timeScope 此排行榜搜索的时间段/范围。
title 此排行榜的可读名称。
userScope 此排行榜搜索的用户范围。

公共函数

LoadScores 根据此排行榜上设置的过滤器加载得分。
SetUserFilter 仅搜索这些用户 ID。
