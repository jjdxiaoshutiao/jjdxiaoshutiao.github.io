CameraType

enumeration

切换到手册

描述

描述不同类型的摄像机。

Unity 编辑器根据用途设置摄像机类型。在项目中选择模型资源时，Inspector 中的预览将通过类型为 Preview 的摄像机渲染。场景层级视图中的摄像机的类型为 Game。在场景层级视图中选择摄像机时显示的预览窗口是所选摄像机的副本，因此类型也是 Game。

变量

Game 用于指示普通的游戏内摄像机。
SceneView 用于指示在 Editor 中渲染场景视图的摄像机。
Preview 用于指示在 Editor 中渲染预览的摄像机。
VR 用于指示在 Editor 中渲染 VR（编辑模式）的摄像机。
Reflection 用于指示渲染反射探针的摄像机。
