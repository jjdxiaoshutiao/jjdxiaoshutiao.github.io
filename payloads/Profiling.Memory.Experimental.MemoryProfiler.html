class in UnityEngine.Profiling.Memory.Experimental

描述

内存性能分析 API 容器类。

此 API 在编辑器和播放器之间是统一的。

静态函数

TakeSnapshot 触发内存快照捕获。
TakeTempSnapshot 触发以 Application.temporaryCachePath 文件夹为目标位置的内存快照捕获。

Events

createMetaData 收集方法可订阅的元数据事件。
