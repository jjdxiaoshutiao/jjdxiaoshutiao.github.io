SystemInfo

class in UnityEngine

切换到手册

描述

访问系统和硬件信息。

使用该类可以确定底层平台和硬件的功能。例如，您可以检查支持的 RenderTexture 格式 ( SupportsRenderTextureFormat )、可用的 CPU 线程数 ( processorCount ) 等。
